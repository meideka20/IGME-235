Missile Command Asteroids

Concept

So let's think about the classic arcade game Asteroids, but then instead of the ship shooting the asteroids, there's a turret at the bottom of the screen that you shoot with. Essentially, this is a mix-up between Asteroids and Missile Command, both of which are old arcade games

Genre

The genre of this game is the same genre as the original Asteroids, a shoot-em-up arcade game.

Platform

This game is going to be only on desktop

Story

The story is simple, you're trying to keep the spaceship from being blown up while also not shooting it in the process

Aesthetics

I'd like this game to have the same aesthetics as the original games Asteroids and Missile Command, so an old arcade-y look

Gameplay

Mechanics

This game will focus on moving a ship around an area, avoiding asteroids, and shooting a turret at those asteroids, avoiding the ship

Control

To control the ship you'll use WASD and to control the turret you'll move the mouse and click to shoot

Player Learning

The player will start off with an instruction screen of how to play, then the first level will have only 1 or 2 asteroids. Then, after destroying all of the asteroids, the next level will start with more, and so on and so forth
